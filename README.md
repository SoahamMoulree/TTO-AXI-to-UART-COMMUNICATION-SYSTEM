# AXI-UART_Communication_Stream_TTO  
This system converts parallel data from an AXI Stream interface to serial UART format and vice versa, featuring:  
* AXI Stream Interface for high-speed parallel data input  
* Synchronous FIFO Buffer (8-byte depth) for data rate adaptation  
* UART Transmitter with configurable baud rate and parity  
* UART Receiver with parity verification and data validation
